# 📱 MobileShop - E-Commerce Platform
A robust Full-Stack E-Commerce web application developed using **ASP.NET MVC** and **SQL Server**. This project focuses on providing a seamless experience for both customers and administrators.

## 🚀 Key Features
* **Admin Dashboard**: Real-time stats for Revenue, Orders, and Users.
* **Order Tracking**: Manage order statuses like Pending, Shipped, and Delivered.
* **PDF Invoices**: Generate and download professional digital invoices using Rotativa.
* **Secure Authentication**: User registration with Password validation and Forgot Password logic.
* **Search Functionality**: Quick product discovery for mobile models.

## 🛠️ Tech Stack
* **Backend**: ASP.NET MVC 5, C#
* **Database**: SQL Server (Entity Framework)
* **Frontend**: Bootstrap 5, Razor Views, CSS3
* **Tools**: Visual Studio 2022, Git/GitHub
